## Scripts

### Web

- Run:
  `npm run web`
- Build:
  `npm run build:web`
- Deploy to GitHub Pages:
  `npm run deploy <GH_USERNAME> <GH_TOKEN>`

### Desktop

- Run:
  `npm run desktop`
- Build for Windows:
  `npm run build:windows`
- Build for Linux:
	`npm run build:linux`

### Shared:

- Run tests:
  `npm test`
